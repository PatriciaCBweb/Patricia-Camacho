APLICACIÓN DE LOGIN

Aplicación de login, con validación de datos, contador de contraseñas erróneas, registro 
y redirección de página en función del rol del usuario.

Realizado con html, Bootstrap y php.

1. Abrir bbdd.php y tablaAdmin.php para crear la base de datos y la tabla que alojará
   la información de usuarios.
2. Abrir usuarios -> index.php
